# 📊 HR Recruitment Dashboard (Power BI)

## 📌 Project Overview

This project is an interactive **HR Recruitment Dashboard** built using **Microsoft Power BI**. It analyzes recruitment data and provides insights into candidate sourcing, screening, interview progress, onboarding, and hiring outcomes.

The dashboard was created as part of my HR internship to strengthen my skills in **Power BI, Excel, HR Analytics, and Data Visualization**.

## 🔒 Dataset Information

**Note:**  
The dataset included in this repository has been **anonymized for portfolio purposes**.

- No candidate names
- No email addresses
- No phone numbers
- No confidential company information
- Candidate IDs are used instead of personal information

The dataset is shared only to demonstrate dashboard development and data analysis skills.
